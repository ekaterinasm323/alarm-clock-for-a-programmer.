# 3. Введение
Будильник для программистов - это будильник, созданный для людей с ненормированным графиком сна. Его главная особенность - интеграция с системой контроля версий.
Назначение продукта: Помогать людям просыпаться, вовремя ложиться спать, а так же не забывать про важные события.

# 4. Примеры использования
1. Установка времени - при нажатии на кнопку "+" или "-" пользователь может установить точное время воспроизведения будильника.
2. Старт будильника - при нажатии на кнопку "Старт" будильник начинает отсчет до установленного времени
3. Остановка будильника - при нажатии на кнопку "Стоп" звук прекращается, будильник ожидает 5 минут
4. Режим "Не беспокоить" - при нажатии на кнопку "Не беспокоить" будильник отключает все звуки
5. Настройка громкости - при нажатии на кнопку "Звук" пользователь регулирует уровень громкости
6. Просмотр статистики - при нажатии на кнопку "Статистика" отображается график, использования будильника
